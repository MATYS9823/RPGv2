# RPGv2

### Install the game

- [Download](https://github.com/MrGg570/RPGv2/archive/refs/heads/main.zip) the code archive and extract it

- Install dependancies:
    ```ssh
    python -m pip install -r requirements.txt
    ```

- Run main python programm:
    ```ssh
    python main.py
    ```
